This repository is the example of [@react-ssr/nestjs-express](https://npm.im/@react-ssr/nestjs-express).

## Usage

```properties
# installation
git clone https://github.com/saltyshiomix/react-ssr-nestjs-starter.git
cd react-ssr-nestjs-starter
npm ci

# development mode
npm run dev

# production mode
npm run build
npm run start
```
